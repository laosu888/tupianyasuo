# 图片压缩工具

一个简单易用的在线图片压缩工具，具有精美的苹果风格界面设计。

## 功能特点

- 支持PNG、JPG等格式图片上传
- 支持自定义压缩比例
- 实时预览压缩前后的图片效果
- 显示压缩前后文件大小对比
- 支持压缩后图片下载
- 纯前端实现，无需后端服务

## 项目结构

```
├── index.html          # 主页面
├── css/               
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # 主要功能实现
└── assets/
    └── icons/         # SVG图标
```

## 技术栈

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- 浏览器原生图片压缩API
